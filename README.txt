Sync WooCommerce products and modules with Odoo via API. Supports bulk and single product import with updates.

== Description ==
Odoo Woo is a WordPress plugin that allows seamless integration between Odoo ERP and WooCommerce.

Features:
* View installed Odoo modules
* Import and update products from Odoo to WooCommerce
* Bulk import all products sequentially with loader and status
* Single import button per product for manual control
* Prevents duplicates using _odoo_id meta
* Fully AJAX-powered
* Works even if products have no SKU

== Installation ==
1. Upload the plugin folder to wp-content/plugins/
2. Activate the plugin from the WordPress admin
3. Configure your Odoo credentials in the plugin settings
4. Go to Odoo Woo menu to import modules/products
